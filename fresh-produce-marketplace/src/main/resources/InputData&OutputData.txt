http://localhost:8080/AgroFarm/customer-api/login

{

4. For getting all products
GET Request :
http://localhost:8080/AgroFarm/product-api/products

5. For getting Product details by Product Id:
GET Request :
http://localhost:8080/AgroFarm/product-api/product/1005

6. For Updating the available quantity of product:
PUT Request :
http://localhost:8080/AgroFarm/product-api/updatequantity/1005
